Game Programming Using HTML 5



HTML5 is the latest and most enhanced version of HTML. Technically, HTML is not a programming language, but rather a markup language. In this tutorial, we will discuss the features of HTML5 and how to use it in practice.

Course Code: CSG4010

LT Course with 3 Credits

What We'll Learn In this Course



  1. Fundamentals of HTML and HTML 5
  2. Javascript fundamentals, variables, constants, loops, operators, functions, objects-classes
  3. game programming with HTML5, Maths, Game Physics
  4. game programming : how to take Inputs, cameras, cursors, dragging
  5. Textures; Time: events, Start at, Pause and Resume, Looped events, etc;
